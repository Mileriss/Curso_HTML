COLSPAN E ROWSPAN

As tabelas HTML podem ter células que se estendem por várias linhas e/ou colunas.

Tabela HTML - Colspan Para fazer uma célula se estender por várias colunas, use o colspanatributo:
Name Age
Jill Smith 43
Eve Jackson 57
Tabela HTML - Linhas Para fazer uma célula se estender por várias linhas, use o rowspanatributo:
Name Jill
Phone 555-1234
555-8745
