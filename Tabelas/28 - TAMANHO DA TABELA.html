TAMANHO DA TABELA

As tabelas HTML podem ter tamanhos diferentes para cada coluna, linha ou toda a tabela.

Use o styleatributo com as propriedades widthou height para especificar o tamanho de uma tabela, linha ou coluna.

Largura da Tabela HTML Para definir a largura de uma tabela, adicione o style atributo ao elemento: Defina a largura da tabela para 100%:
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
Largura da Coluna da Tabela HTML Para definir o tamanho de uma coluna específica, adicione o style atributo em um ou elemento: Defina a largura da primeira coluna para 70%:
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
Altura da linha da tabela HTML Para definir a altura de uma linha específica, adicione o style atributo em um elemento de linha da tabela: Defina a altura da segunda linha para 200 pixels:
