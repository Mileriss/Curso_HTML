USANDO EMOJI

Emojis são caracteres do conjunto de caracteres UTF-8: 😄 😍 💗

O que são Emojis? Os emojis parecem imagens ou ícones, mas não são. São letras (caracteres) do conjunto de caracteres UTF-8 (Unicode). O atributo do conjunto de caracteres HTML Para exibir uma página HTML corretamente, um navegador da Web deve conhecer o conjunto de caracteres usado na página. Isso é especificado na tag: Caracteres UTF-8 Muitos caracteres UTF-8 não podem ser digitados em um teclado, mas sempre podem ser exibidos usando números (chamados números de entidade): A é 65 B é 66 C é 67 Exemplo

I will display A B C

I will display A B C

O elemento define o conjunto de caracteres. Os caracteres A, B e C são exibidos pelos números 65, 66 e 67. Para permitir que o navegador entenda que você está exibindo um caractere, você deve iniciar o número da entidade com e terminá-lo com ; (ponto e vírgula). Caracteres Emoji Os emojis também são caracteres do alfabeto UTF-8: 😄 é 128516 😍 é 128525 💗 é 128151
