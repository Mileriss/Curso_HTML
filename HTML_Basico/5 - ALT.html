ALT

O atributo alt: O alt atributo obrigatório para a tag especifica um texto alternativo para uma imagem, se a imagem por algum motivo não puder ser exibida.
Isso pode ser devido a uma conexão lenta, um erro no srcatributo ou se o usuário usar um leitor de tela.
