RESUMO DOS ATRIBUTOS

Todos os elementos HTML podem ter atributos:

O href atributo de especifica o URL da página para a qual o link vai

O src atributo de especifica o caminho para a imagem a ser exibida

Os atributos width e fornecem informações de tamanho para imagens height

O alt atributo of fornece um texto alternativo para uma imagem

O style atributo é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e mais

O lang atributo da tag declara o idioma da página da Web

O title atributo define algumas informações extras sobre um elemento
