COMENTÁRIOS

Comentários HTML não são exibidos no navegador, mas podem ajudar a documentar seu código-fonte HTML.

Você pode adicionar comentários à sua fonte HTML usando a seguinte sintaxe: Com os comentários, você pode colocar notificações e lembretes em seu código HTML:

This is a paragraph.

Ocultar conteúdo:

Isso é um paragráfo.

Isso é um paragráfo.

Ocultar uma seção do código HTML:

This is a paragraph.

This is a paragraph too.

Ocultar uma parte de um parágrafo:

This is a paragraph.
