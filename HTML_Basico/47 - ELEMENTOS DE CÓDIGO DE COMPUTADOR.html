ELEMENTOS DE CÓDIGO DE COMPUTADOR

O HTML contém vários elementos para definir a entrada do usuário e o código do computador.

x = 5; y = 6; z = x + y; HTML para entrada de teclado O elemento HTML é usado para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo Defina algum texto como entrada de teclado em um documento:

Save the document by pressing Ctrl + S

Resultado: Save the document by pressing Ctrl + S HTML para saída do programa O elemento HTML é usado para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo Defina algum texto como saída de amostra de um programa de computador em um documento:

Message from my computer:

File not found.
Press F1 to continue

Resultado: Message from my computer: File not found. Press F1 to continue HTML para código de computador O elemento HTML é usado para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo Defina algum texto como código de computador em um documento: x = 5; y = 6; z = x + y; Resultado: x = 5; y = 6; z = x + y; Observe que o elemento não preserva espaços em branco extras e quebras de linha. Para corrigir isso, você pode colocar o elemento dentro de um 
elemento:
                    
                    Exemplo
                    
                    
                    x = 5;
                    y = 6;
                    z = x + y;
                    
                    


                    Resultado:
                    x = 5;
                    y = 6;
                    z = x + y;
                    

                    HTML  para variáveis
                        O elemento HTML é usado para definir uma variável na programação ou em uma expressão matemática. O conteúdo interno geralmente é exibido em itálico.
                        
                        Exemplo
                        Defina algum texto como variáveis em um documento:
                        
The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.


                        Resultado:
                        The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.
