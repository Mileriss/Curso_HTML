ATRIBUTOS DE CLASSE

O atributo HTML classé usado para especificar uma classe para um elemento HTML. Vários elementos HTML podem compartilhar a mesma classe.

Usando o atributo de classe O classatributo é frequentemente usado para apontar para um nome de classe em uma folha de estilo. Ele também pode ser usado por um JavaScript para acessar e manipular elementos com o nome de classe específico. No exemplo a seguir temos três
elementos com um classatributo com o valor "city". Todos os três
elementos serão estilizados igualmente de acordo com a .city definição de estilo na seção principal:

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.

No exemplo a seguir temos dois elementos com um classatributo com o valor "note". Ambos os elementos serão estilizados igualmente de acordo com a .note definição de estilo na seção head:

My Important Heading

This is some important text.
