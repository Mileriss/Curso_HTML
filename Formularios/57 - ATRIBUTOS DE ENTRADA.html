Um campo de entrada somente leitura não pode ser modificado (no entanto, um usuário pode tabular até ele, realçá-lo e copiar o texto dele). O atributo maxlength O maxlength atributo de entrada especifica o número máximo de caracteres permitidos em um campo de entrada. Nota: Quando a maxlengthé definido, o campo de entrada não aceitará mais do que o número especificado de caracteres. No entanto, esse atributo não fornece nenhum feedback. O atributo step O atributo input stepespecifica os intervalos de números legais para um campo de entrada. Exemplo: if step="3", os números legais podem ser -3, 0, 3, 6, etc. Dica: Este atributo pode ser usado junto com os atributos max e min para criar um intervalo de valores válidos.
