FORMULÁRIOS

Um formulário HTML é usado para coletar a entrada do usuário. A entrada do usuário é mais frequentemente enviada a um servidor para processamento.

O elemento
O elemento HTML é usado para criar um formulário HTML para entrada do usuário: . form elements .
O
elemento é um contêiner para diferentes tipos de elementos de entrada, como: campos de texto, caixas de seleção, botões de opção, botões de envio, etc. Todos os diferentes elementos de formulário são abordados neste capítulo: Elementos de formulário HTML . O elemento O elemento HTML é o elemento de formulário mais usado. Um elemento pode ser exibido de várias maneiras, dependendo do type atributo. aqui estão alguns exemplos: Tipo Descrição Exibe um campo de entrada de texto de linha única Exibe um botão de opção (para selecionar uma das muitas opções) Exibe uma caixa de seleção (para selecionar zero ou mais de muitas opções) Exibe um botão enviar (para enviar o formulário) Exibe um botão clicável Todos os diferentes tipos de entrada são abordados neste capítulo: Tipos de entrada HTML .
