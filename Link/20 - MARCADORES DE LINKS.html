MARCADORES DE LINKS

Criar um marcador em HTML Os marcadores podem ser úteis se uma página da Web for muito longa. Para criar um marcador - primeiro crie o marcador e, em seguida, adicione um link a ele. Quando o link for clicado, a página rolará para baixo ou para cima até

Primeiro, use o idatributo para criar um marcador:

Chapter 4

Em seguida, adicione um link ao marcador ("Pular para o Capítulo 4"), de dentro da mesma página: Jump to Chapter 4 Você também pode adicionar um link para um favorito em outra página: Jump to Chapter 4 Use o id atributo (id=" value ") para definir marcadores em uma página Use o href atributo (href="# value ") para vincular ao marcador
