LISTAS ORDENADAS

A tag HTML

    define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

    Lista HTML ordenada Uma lista ordenada começa com a
      tag. Cada item da lista começa com a
    1. tag. Os itens da lista serão marcados com números por padrão:
      1. Coffee
      2. Tea
      3. Milk
      Lista HTML ordenada - O atributo de tipo O typeatributo da
        tag, define o tipo do marcador de item da lista: Type Description type="1" The list items will be numbered with numbers (default) type="A" The list items will be numbered with uppercase letters type="a" The list items will be numbered with lowercase letters type="I" The list items will be numbered with uppercase roman numbers type="i" The list items will be numbered with lowercase roman numbers Números:
        1. Coffee
        2. Tea
        3. Milk
        Letras maiúsculas:
        1. Coffee
        2. Tea
        3. Milk
        Letras minúsculas:
        1. Coffee
        2. Tea
        3. Milk
        Números romanos maiúsculos:
        1. Coffee
        2. Tea
        3. Milk
        Números romanos minúsculos:
        1. Coffee
        2. Tea
        3. Milk
        Contagem da lista de controle Por padrão, uma lista ordenada começará a contar a partir de 1. Se você quiser começar a contar a partir de um número especificado, você pode usar o startatributo:
        1. Coffee
        2. Tea
        3. Milk
        Listas HTML aninhadas As listas podem ser aninhadas (lista dentro da lista):
        1. Coffee
        2. Tea
          1. Black tea
          2. Green tea
        3. Milk
        Use o elemento HTML
          para definir uma lista ordenada Use o atributo HTML type para definir o tipo de numeração Use o elemento HTML
        1. para definir um item de lista As listas podem ser aninhadas Os itens da lista podem conter outros elementos HTML
