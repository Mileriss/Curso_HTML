HTML também suporta listas de descrição.

Listas de descrição HTML Uma lista de descrição é uma lista de termos, com uma descrição de cada termo. A
tag define a lista de descrição, a tag define o termo (nome) e a tag descreve cada termo:
Coffee
- black hot drink
Milk
- white cold drink
