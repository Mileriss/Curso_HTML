O que é HTML Canvas? O elemento HTML é usado para desenhar gráficos, em tempo real, via JavaScript. O elemento é apenas um contêiner para gráficos. Você deve usar JavaScript para realmente desenhar os gráficos. O Canvas tem vários métodos para desenhar caminhos, caixas, círculos, texto e adicionar imagens.
